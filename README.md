# Introducing the daily_planner gem


## Creating a new daily-planner.txt file

    require 'daily_planner'

    dp = DailyPlanner.new
    dp.save

Output:

<pre>
daily-planner.txt
=================

Friday, 11th Dec
----------------

Tomorrow
--------

</pre>
